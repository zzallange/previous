다크모드 코드적용

IE 적용

-a링크로 btn-darkmode 버튼

-버튼클릭시 body에 dark-theme클래스 적용

-body의 클래스가 dark-theme 추가시 각각의 color 적용 .title {color:#000} body.dark-theme .title {color:#fff}

최신 CSS3적용

-input:checked를 활용한 toggle 버튼

-data-theme: dark / light

-변수를 정의한 상위요소 :root css3적용 :root{ --title-color:#000 } [data-theme="dark"]{ --title-color:#fff } .title {color: var(--title-color)}
