Dark mode 적용

적용 사례 및 이유
끊임없는 혁신기술 애플,구글,삼성 등 대기업에서 주력하는 상품들은 모두 OLED 화면을 갖추고 있다.
이는 LCD 디스플레이와 달리 백라이트가 필요하지 않기 때문에 배터리 수명에 좋다.
OLED는 다크화면시 시각 이미지를 구성하는 픽셀이 꺼지면서 에너지 소비가 적다.
계속해서 주요 IT회사들 네이버등 다크모드를 적용 중에 있다.
진행 중 접근성 향상 색상 대비 접근성 표준을 지켜 시력이 약한 사용자도 사용 할 수 있다.
장시간 스크린을 볼때 나타나는 눈의 피로도를 줄여줌으로써, 사용자 경험까지 더욱 향상시킨다고 알려져있다.

다크모드 코드적용

IE 적용

-a링크로 btn-darkmode 버튼

-버튼클릭시 body에 dark-theme클래스 적용

-body의 클래스가 dark-theme 추가시 각각의 color 적용 .title {color:#000} body.dark-theme .title {color:#fff}

최신 CSS3적용

-input:checked를 활용한 toggle 버튼

-data-theme: dark / light

-변수를 정의한 상위요소 :root css3적용 :root{ --title-color:#000 } [data-theme="dark"]{ --title-color:#fff } .title {color: var(--title-color)}
